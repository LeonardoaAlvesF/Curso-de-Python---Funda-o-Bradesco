Repositório com anotações, exercícios e projetos desenvolvidos durante o curso básico de Python oferecido gratuitamente pela **Fundação Bradesco**.

## Sobre o curso

O curso *Programar em Python Básico* é oferecido gratuitamente pela Escola Virtual da Fundação Bradesco. Aborda os fundamentos da linguagem Python, desde a instalação do ambiente até estruturas de controle, funções e manipulação de dados.

## Conteúdo do repositório

- `modulos/` — Exercícios e exemplos por módulo do curso
- `projetos/` — Projetos práticos desenvolvidos ao longo do curso
- `anotacoes/` — Resumos e anotações pessoais sobre os temas

## Tópicos abordados

- Introdução ao Python e configuração do ambiente
- Variáveis, tipos de dados e operadores
- Estruturas condicionais (`if`, `elif`, `else`)
- Estruturas de repetição (`for`, `while`)
- Funções e modularização
- Listas, tuplas, dicionários e conjuntos
- Manipulação de strings
- Tratamento de erros e exceções
- Introdução a arquivos

## Como executar

Certifique-se de ter o Python 3 instalado. Para rodar qualquer script:

```bash
python nome_do_arquivo.py
```
